YOLO-based traffic detection with Arduino control

Detect pedestrians, vehicles and ambulances on four lanes using Ultralytics YOLO and control lane lights on an ESP32/Arduino via serial and WiFi requests.

Setup

Non-destructive steps (keep code unchanged)
  1. Create files in GitHub web UI and paste your original scripts exactly as they are.
  2. Add models/best (3).pt to models/ if you want to store it in the repo (consider .gitignore if too large).
  3. Install dependencies locally or on the machine running detection: 
    pip install -r requirements.txt
  4. Open and flash microcontroller/traffic_control.ino to your ESP32 using Arduino IDE.
  5. Connect the ESP32 to your PC and run the detection scripts; they will communicate via serial (COM port).
